================================================================================
🔧 TROUBLESHOOTING
================================================================================

"Python not found"
→ Install Python from https://www.python.org/
→ Check: python --version

"FFmpeg not found"
→ Install FFmpeg (see STEP 2 above)
→ Verify: ffmpeg -version

"ModuleNotFoundError: No module named 'cv2'"
→ Run: pip install -r requirements.txt
→ Or: pip install opencv-python

"Address already in use"
→ Another app is using port 5000
→ Either close it or change port in code (line: app.run(port=5001))

"Video generation takes too long"
→ Normal! First time may take 2-5 minutes
→ Depends on computer speed
→ Subsequent videos are faster

"Audio quality is robotic"
→ That's the free TTS service
→ Upgrade to paid TTS for better quality (optional)

Video won't play
→ Download a video player (VLC media player)
→ Or use: Windows Media Player, QuickTime

================================================================================
💡 TIPS & TRICKS
================================================================================

1. Image Preparation
   - Use high-quality images (at least 720p)
   - 16:9 aspect ratio works best
   - Simple backgrounds work better
   - Remove clutter around product

2. Features
   - Use 3-5 clear, short features
   - Avoid technical jargon
   - Focus on benefits, not specs
   - Example: "30-hour battery" not "high capacity Li-ion"

3. Call to Action
   - Short and punchy
   - Action-oriented verbs: Order, Buy, Get, Discover
   - Examples: "Buy Now", "Order Today", "Get 50% Off"

4. Video Usage
   - Share on TikTok (great engagement)
   - Post on Instagram Reels
   - Use as YouTube ads
   - Add to your website
   - Email to customers

📈 PERFORMANCE TIPS
================================================================================

If video generation is slow:

1. Use smaller images (1920x1080 instead of 4K)
2. Close other applications
3. Use simple, non-transparent PNGs
4. Reduce number of features (use 3 max)

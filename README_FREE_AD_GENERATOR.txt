Common Issues:

"Python not found"
→ Install Python from https://python.org/

"FFmpeg not found"
→ Install FFmpeg (see STEP 1 above)

"Import error"
→ Run: pip install -r requirements.txt

"Address already in use"
→ Close other apps or change port in code

Video won't play
→ Try VLC media player (free)

Start Server:

  Press Ctrl+C in terminal

  Edit: ad_generator_backend.py
  Line: app.run(debug=True, port=5000)
  Change 5000 to another number
